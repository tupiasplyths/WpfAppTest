## Implementation Details

### Behavior
- When user presses Escape key, window minimizes and hides from taskbar
- Tray icon appears in system notification area
- Single click on tray icon toggles window visibility
- Right-click shows context menu with "Restore" and "Exit" options
- When window is restored, it shows as maximized full-screen

### Technical Notes
- Used System.Windows.Forms.NotifyIcon for tray functionality
- Wrapped in custom SystemTrayIcon class for cleaner API
- Icon loaded from embedded resource (Resources/icon.ico)
- Fallback to application icon if custom icon not found
- Proper disposal of tray icon resources on window closing

### Namespace Ambiguity Issues
- Adding `<UseWindowsForms>true</UseWindowsForms>` introduced namespace conflicts
- Fixed by fully qualifying type names in affected files
- Affected types: Application, Point, KeyEventArgs, MouseEventArgs, TextBox, Cursors
